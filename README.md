# AWS Starter Guide

An interactive learning guide for AWS, covering core services with hands-on labs, quizzes, cheatsheets, and progress tracking. Content is in Vietnamese.

## Prerequisites

- Node.js 20+
- pnpm

## Setup

```bash
git clone <repo-url>
cd aws-starter-guide
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
pnpm build
pnpm start
```

## Tech Stack

- **Next.js 16** (App Router, Turbopack)
- **React 19**
- **TypeScript**
- **Tailwind CSS v4**
- **Radix UI** (accessible primitives)
- **Framer Motion** (animations)
- **Lucide React** (icons)
- **next-themes** (dark/light mode)
- **Fuse.js** (fuzzy search)
